# ¿Cómo usar?

1. Clonar repositorio

```shell
git clone https://github.com/tutosrive/NBAI.git
```

2. Descargar/Verificar cambios

```shell
git pull -u origin main
```

3. Añadir cambios

```shell
# Ejecute uno a uno
git add .
git commit -m "Add here your message"
```
